# BESS Telemetry Interface Documentation

Welcome to the Bess Telemetry Interface Documentation. Here you will have developer resources to let you run this codebase. Test it and replicate it on your machine.

The directory structure has two main directories, dev and user. Dev contains information for developers. Setting up your development environment, how to run tests and the development build of the software.
